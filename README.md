# Professional Survey Analysis Dashboard

## Project Overview

Developed an interactive Power BI dashboard to analyze survey responses from over 1,000 data professionals. The dashboard provides insights into salary trends, job satisfaction, work-life balance, and technology preferences.

## Tools Used

* Power BI
* Power Query
* Data Visualization

## Dashboard Features

* Salary analysis by job role
* Job satisfaction and work-life balance insights
* Programming language preferences
* Interactive filters and visualizations
* Survey response analysis

## Dataset

* Professional survey dataset containing over 1,000 responses
* Includes salary, job role, satisfaction, and technology-related information

## Skills Demonstrated

* Data Cleaning
* Data Transformation
* Data Analysis
* Dashboard Development
* Data Visualization
* Business Intelligence

## File Information

The PBIX file contains the dataset, data transformations, and the final interactive dashboard used for analysis and reporting.
